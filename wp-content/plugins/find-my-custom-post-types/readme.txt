Finds and displays the names of the custom post types running on your website.   

== Description ==

The Find My Custom Post Types plugin displays the names of any custom post types created by your theme or plugins. It will also indicate which of the custom post types are publicly queryable.


== Installation ==

1. Install the Find My Custom Post Types plugin either via the WordPress plugin directory, or by uploading the files to the /wp-content/plugins/ directory. 
2. Activate the plugin in the Plugins page in WordPress. 
3. The display page for Find My Custom Post Types can be found in the admin menu under Tools > Find My CPTs.  


== Find My Custom Post Types Features ==

* Displays any custom post types registered on your website.
* Indicates which custom post types are publicly queryable. 


== Frequently Asked Questions ==

=  Where does Find My Custom Post Types display my custom post types? =

Find My Custom Post Types can be found in the admin menu under Tools > Find My CPTs.
